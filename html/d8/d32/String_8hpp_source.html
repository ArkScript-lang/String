221  /**
222  * @brief Format a string using snprintf
223  * @details The format is held by the string itself, then the arguments are passed to replace the format specifiers.
224  * Since this function is templated, a common format specifier `%%` replaces (nearly) all the other known ones.
225  * Thus, when there are no argument left, this function doesn't go on an undefined behaviour trying to read
226  * arguments which weren't given, it will just continue to copy the rest of the string and leave the other format
227  * specifiers in the string (if any) untouched.
228  * Since the common format specifier is `%%`, writing a single `%` is just adding a `%` to the string.
229  * For integers, there is a second specifier `%x` to write the number in hexadecimal.
230  * The instance is modified and size is changed, as well as the internal buffer.
231  *
232  * @tparam Args
233  * @param n Maximum number of bytes to use for the new string
234  * @param args Arguments to format
235  * @return String&
236  */
237  template<typename... Args>
238  String& format(unsigned n, Args&&... args);
